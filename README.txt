Backroom v1.02 — Admin record preview

Upload these three files to the root of the Backroom-site repository:
- admin.html
- admin.js
- admin-record-preview.html

What changed:
- Click inside any venue or event row, then choose “View Record”.
- A separate draft preview window opens, showing the selected record in a Backroom-style public listing layout.
- The preview uses the current admin draft record, so unsaved row edits are included once you leave the cell.
- The left menu now includes “Voting Admin”, which opens ratings-admin.html.
